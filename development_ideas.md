# Идеи для дальнейшего развития проекта

## Геймплей
- Система достижений (убить N врагов, найти N сундуков, пройти N уровней)
- Возможность крафта предметов из ресурсов
- Торговец между уровнями (продажа/покупка предметов)
- Разные типы стрел/снарядов (огненные, ледяные, отравляющие)
- Умения/скиллы, открываемые с уровнем

## Графика и UI
- Туман войны (Fog of War) — неисследованные участки карты скрыты
- Сглаживание/тени для тайлов карты
- Анимации для атак (взмах меча, полёт стрелы)
- Индикатор получения урона (экран краснеет)
- Подсказки при наведении на предметы в инвентаре

## Карта и мир
- Несколько уровней/этажей подземелья (лестницы вниз)
- Комнаты с разными темами (лава, лес, лёд, пустыня)
- Интерактивные объекты (двери, рычаги, ловушки)
- Генерация деревень/поселений на поверхности
- Подземные реки и мосты

## Технические улучшения
- Оптимизация производительности для больших карт (LOD, фрустум каллинг)
- Модульная система предметов и эффектов (компонентный подход)
- Поддержка мобильного управления (touch)
- Сетевая игра (co-op)
- Редактор карт в игре
