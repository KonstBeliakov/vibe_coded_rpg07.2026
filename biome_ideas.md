# Идеи для биомов

## Существующие биомы

### 1. Обычный (Normal) — `BIOME_NORMAL`
- **Текущее состояние**: Темно-серый пол (#2a2a2a), обычные стены
- **Идеи развития**:
  - Добавить редкие декоративные элементы (камни, трещины на полу)
  - Иногда встречаются "сокровищницы" — маленькие комнаты с сундуком, окруженные стенами
  - Добавить "тропы" — полосы более светлого пола, указывающие направление к ближайшей безопасной зоне

### 2. Мшистый (Mossy) — `BIOME_MOSSY`
- **Текущее состояние**: Темно-зеленый пол (#1a3a1a), цветы (восстанавливают 5 HP)
- **Идеи развития**:
  - Добавить "грибные поляны" — большие грибы, которые при сборе дают временные баффы
  - Ядовитые цветы (красные) — наносят урон при касании, но из них можно добыть essence
  - Добавить мшистые стены — стены, покрытые мхом, которые можно разрушить киркой
  - Увеличить плотность цветов в центре биома, уменьшить к краям

### 3. Паутинный (Web) — `BIOME_WEB`
- **Текущее состояние**: Синевато-темный пол (#2a2a3a), паутина замедляет до 20% скорости
- **Идеи развития**:
  - Добавить "паучьи гнезда" — структуры из паутины, в которых спавнятся пауки (новый тип врага)
  - Коконы — висят с потолка, при разрушении выпадает лут (монеты, ресурсы)
  - Королева пауков — мини-босс, появляется в центре больших паутинных зон
  - Паутина со временем может восстанавливаться, если её не уничтожить полностью

### 4. Лавовый (Lava) — `BIOME_LAVA`
- **Текущее состояние**: Красновато-темный пол (#3a1a1a), лавовые озера наносят урон
- **Идеи развития**:
  - Добавить "огненных элементалей" — враги, которые не получают урон от лавы
  - Мосты через лаву — узкие проходы из камня, по которым можно перебраться
  - Лавовые трещины — случайные разломы, из которых вылетают искры (наносят урон)
  - В центре лавового биома может быть "огненный алтарь" — дает временный иммунитет к огню
  - Редкая руда "обсидиан" — добывается только в лавовом биоме, используется для крафта мощного оружия

## Новые биомы (идеи)

### 5. Ледяной (Ice) — `BIOME_ICE`
- **Описание**: Холодный биом с голубоватым полом
- **Цвет пола**: #1a2a3a (темно-синий)
- **Особенности**:
  - Скользкий пол — игрок продолжает движение по инерции (не может резко остановиться)
  - Сосульки на стенах — при падении наносят урон
  - Ледяные враги — медленные, но с большим запасом здоровья
  - Замерзшие озера — можно провалиться под лед
  - Кристаллы льда — ресурс для крафта ледяных стрел/снарядов

### 6. Пустынный (Desert) — `BIOME_DESERT`
- **Описание**: Жаркий песчаный биом
- **Цвет пола**: #3a3a1a (желтовато-темный)
- **Особенности**:
  - Песчаные бури — периодически снижают видимость (уменьшают радиус обзора)
  - Зыбучие пески — замедляют и постепенно затягивают игрока
  - Скорпионы — новые враги, атакуют ядом
  - Оазисы — маленькие безопасные зоны с водой и пальмами
  - Древние руины — структуры с повышенным шансом найти редкий лут

### 7. Магический (Magic) — `BIOME_MAGIC`
- **Описание**: Таинственный биом с магической аурой
- **Цвет пола**: #1a1a3a (фиолетово-темный)
- **Особенности**:
  - Мана-фонтаны — восстанавливают здоровье и дают временный бафф к урону
  - Телепорты — перемещают игрока в другое место биома
  - Магические барьеры — стены, которые можно пройти только имея определенный предмет
  - Призраки — враги, которые проходят сквозь стены
  - Кристаллы маны — ресурс для крафта магических предметов

### 8. Болотный (Swamp) — `BIOME_SWAMP`
- **Описание**: Темное болото с ядовитой атмосферой
- **Цвет пола**: #1a2a1a (темно-зеленый с оттенком)
- **Особенности**:
  - Ядовитые испарения — постепенно отравляют игрока (тик урона каждые 3 секунды)
  - Трясина — замедляет и не дает использовать рывок
  - Светлячки — указывают путь к безопасным зонам
  - Водяные змеи — враги, атакующие из воды
  - Лечебные травы — растут только в болоте, можно крафтить мощные зелья

### 9. Кристальный (Crystal) — `BIOME_CRYSTAL`
- **Описание**: Биом с огромными светящимися кристаллами
- **Цвет пола**: #2a1a3a (фиолетово-темный)
- **Особенности**:
  - Кристаллы освещают область вокруг себя (уменьшают туман войны)
  - Осколки кристаллов — при разрушении наносят урон осколками
  - Кристаллические големы — новые враги с высоким сопротивлением к урону
  - Цветные кристаллы — каждый цвет дает разный бафф при сборе
  - Ресурс "кристаллическая пыль" — используется для улучшения оружия

### 10. Подземное озеро (Underground Lake) — `BIOME_LAKE`
- **Описание**: Большие подземные водоемы
- **Цвет пола/воды**: #1a2a3a (вода)
- **Особенности**:
  - Вода — замедляет движение, но можно плавать
  - Подводные пещеры — скрытые проходы под водой
  - Водные враги — атакуют только в воде
  - Островки — безопасные зоны посреди воды
  - Жемчуг — редкий ресурс на дне озера

## Общие идеи для развития биомов

### Переходные зоны
- Между биомами должны быть плавные переходы (2-3 клетки смешанного типа)
- На границе биомов могут появляться уникальные враги

### Сезонность/цикличность
- Биомы могут менять свои свойства в зависимости от "глубины" (расстояния от центра)
- Например, на большой глубине лавовый биом становится более активным (больше озер)

### Ресурсная система
- Каждый биом должен давать уникальный ресурс
- Ресурсы нужны для крафта предметов, характерных для этого биома
- Редкие ресурсы можно обменять у торговца на уникальные предметы

### Визуальное разнообразие
- Добавить декоративные элементы (сталактиты, корни, грибы, камни)
- Разные типы стен для каждого биома (цвет, текстура)
- Анимированные элементы (мерцание кристаллов, пузырьки в лаве, пар)

### Сложность
- Чем дальше от центра, тем агрессивнее биомы
- На большой глубине биомы могут комбинироваться (например, лава + кристаллы)
