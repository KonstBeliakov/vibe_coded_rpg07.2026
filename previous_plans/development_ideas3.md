# Идеи для дальнейшего развития проекта

## Геймплей
- Система крафта предметов из ресурсов (сундуки дают не только оружие, но и материалы)
- Разные типы зелий (невидимость, регенерация, усиление атаки)
- Умения/скиллы, открываемые с уровнем (щит, рывок, огненный шар)
- Система редкости предметов (обычный, редкий, легендарный) с визуальными эффектами
- Ежедневные задания/испытания

## Графика и UI
- Анимации для атак (взмах меча, полёт стрелы с эффектами)
- Индикатор получения урона (экран краснеет)
- Подсказки при наведении на предметы в инвентаре (tooltip)
- Погодные эффекты (дождь, снег, туман)
- Анимированные спрайты для врагов (ходьба, атака, смерть)

## Карта и мир
- Несколько уровней/этажей подземелья (лестницы вниз)
- Комнаты с разными темами (лава, лес, лёд, пустыня)
- Интерактивные объекты (двери, рычаги, ловушки)
- Подземные реки и мосты
- Секретные комнаты с редким лутом

## Технические улучшения
- Оптимизация производительности для больших карт (LOD, фрустум каллинг)
- Поддержка мобильного управления (touch)
- Сетевая игра (co-op)
- Редактор карт в игре
- Система модов/плагинов
